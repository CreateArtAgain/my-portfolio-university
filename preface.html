ทำไมถึงอยากเป็น “Mobile Application Developer”

ในตอนที่คนเรายังเด็กส่วนมาก จะมีความฝันที่อยากจะเป็น บางคนอยากจะเป็นครูบางคนอยากจะเป็นหมอ ส่วนบางคนอยากจะเป็นนักออกแบบ ทั้งนี้ขึ้นอยู่กับแรงบันดาลใจจากบุคคลรอบข้างและประสบการณ์ของแต่ละคน
สำหรับผมทุกช่วงเวลาที่ผ่านมา ความฝันของผมก็ได้เปลี่ยนแปลงไปตลอดกาลในทุกช่วงเวลาที่ผ่านไป ในช่วงที่ผมยังเป็นเด็ก ผมอยากเป็นหมอ เพราะเป็นอาชีพที่มีเกียรติ และได้รักษาผู้คน ต่อมาเมื่อผมขึ้นชั้นมัธยมตอนต้นผมได้ลองทำอาหาร ทำให้ผมอยากเป็นเชฟ แต่ก็ไม่นานเมื่อครั้งมัธยมศึกษาปีที่3 ครูแนะแนวให้ทำงานเรื่องอาชีพในฝัน นี่เป็นครั้งแรกที่ผมได้สืบค้นข้อมูลในโลกอินเทอร์เน็ต ทำให้ผมเปลี่ยนใจ อยากเป็นวิศวโยธา เพราะเท่ดี แต่เมื่อผลขึ้นมัธยมศึกษาปีที่4 ผมได้ทำโครงงานวิชาไอเอส โดยเลือกหัวข้อไลน์บอททำให้ผมได้ศึกษาเรื่องนี้มาระยะหนึ่ง ซึ่งทำให้ผมสนใจมาก ผมใช้เวลาว่างในช่วงมัธยมศึกษาปีที่4 เพื่อศึกษาว่าอยากเป็นอะไร นั่นคือการเป็นวิศวกรรมคอมพิวเตอร์ แต่พอขึ้นมัธยมศึกษาปีที่5 ผมก็ได้คิดเรื่องอาชีพที่อยากจะเป็นไหม่ โดยเริ่มมองถึงอนาคตขึ้นมาว่าผมอยากเป็นอะไร ผมมีจุดเด่นอะไร ผมชอบอะไร และสนใจอะไร สิ่งที่ผมอยากเป็นแน่นอนว่าต้องเป็นอาชีพด้านเทคโนโลยีแล้วจุดเด่นผมละ จุดเด่นผมก็คือผมมีทักษะการค้นหาข้อมูล การวิเคราะห์ข้อมูล การเรียนรู้สิ่งใหม่ๆ การแก้ปัญหาเฉพาะหน้า และการประยุกต์ใช้สิ่งต่างๆได้ ส่วนสิ่งที่ผมชอบคือ การคำนวณ และผมสนใจเกี่ยวกับการสร้างเกม แอพพลิเคชั่น รวมถึงด้านข้อมูลและไซเบอร์ ด้วยเหตุผลที่กล่าวมาทั้งหมดนี้ ทำให้ผมอยากเป็น Mobile Application Developer เพราะผมสนใจด้านนี้ที่สุด
อาชีพ Mobile Application Developer เป็นหนึ่งในอาชีพ ที่ต้องการมากที่สุดในประเทศ โดยศึกด้านวิทยาศาสตร์คอมพิวเตอร์ และเรียนรู้การใช้ภาษาโปรแกรมที่เหมาะสมกับแพลตฟอร์มที่ต้องการพัฒนา นอกจากนี้ ยังควรมีความเข้าใจเกี่ยวกับการออกแบบ UI/UX การทำงานของ API การทดสอบและแก้ไขข้อผิดพลาด และการปรับปรุงแอปพลิเคชันให้มีประสิทธิภาพ อีกด้วย ทั้งนี้ผมคิดว่าอาชีพนี้เหมาะกับผมมากที่สุดในปัจจุบันและอนาคต เป็นอาชีพที่จะทำให้ผมประสบความสำเร็จ และพร้อมที่จะเรียนรู้สิ่งต่างๆเพราะผมเชื่อว่า “หากเรายังรักและชอบอาชีพนี้เหมือนกับการปลูกต้นไม้ต้องมีสักวันที่ใบไม้จะผลิใบออกผล เฉกเช่นเดียวกับเราที่จะประสบความสำเร็จในสายอาชีพนี้”
